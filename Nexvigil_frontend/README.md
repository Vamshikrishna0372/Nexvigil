# Nexvigil | AI Surveillance Platform

Nexvigil is an advanced AI-powered real-time surveillance platform.

## Features
- Real-time video stream processing
- AI-based object detection
- Rule-based alerts processing
- Professional dashboard

## How to run locally

### Prerequisites
- Node.js & npm installed

### Steps

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the frontend
cd Nexvigil/Nexvigil_frontend

# Install dependencies
npm install

# Start the frontend dev server
npm run dev
```

For the backend, please see the backend documentation or refer to `../RUN_COMMANDS.md`.
